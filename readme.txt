Push notifications from your website or blog via StreamIn

== Description ==

StreamIn plugin enables you to stream notifications from your WordPress blog or website to
your email, phone, Google Chrome extension, GTalk and other StreamIn delivery channels.

For more information visit [http://streamin.io]

== Installation ==

Before you begin, please make sure jQuery is included (should be WP default). If 
you are by chance using Internet Explorer 6, please do not install this plugin or,
even better, stop using Internet.

= Installation steps =

1. Upload 'streamin' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it. Enjoy a full IDE inside Wordpress!
